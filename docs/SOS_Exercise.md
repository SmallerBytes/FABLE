# HOLLOW — SOS Mission Command Exercise

## Purpose

Two-person experiential for **Squadron Officer School**-style mission command training:

- Incomplete information (fog of war)
- Shared understanding built over voice
- Disciplined initiative under threat
- Signature / EMCON management (scans, movement, **speech**)

This is a **fictional** archive scenario. No real unit OPSEC.

## Roles

| Role | Position | Sees | Does |
|---|---|---|---|
| **Operator** | Quest 2 VR (or desktop) | LiDAR returns only | Move, scan, pick up fuses, seat generator, extract |
| **Mission Director (MD)** | Laptop / printout | [Mission Map PDF](../game/assets/HOLLOW_Mission_Map.pdf) / [printable map](../game/map-print.html) | Routes Operator by voice; states intent; does **not** see live Custodian |

Optional third: **Instructor / Safety Observer** — stop word, reset, debrief.

## Commander's intent (read aloud)

> Recover three critical components, restore power to the egress route, and extract the operator. Minimize detectable activity. If compromised, break contact to designated **green harbors** and reorient. Success is extraction with mission assets.

## Setup

1. Print or open the Mission Map for the MD (`game/assets/HOLLOW_Mission_Map.pdf` or `game/map-print.html` → Print → Save as PDF).
2. Operator opens `https://smallerbytes.github.io/FABLE/game/` (or local host) in **Quest Browser**.
3. Allow **microphone** when prompted (speech raises noise).
4. Establish a voice channel (room talk or headset).
5. MD states intent; Operator **backbriefs** the first route.

## Game systems used in the exercise

- **Harbor green LiDAR** — acoustic safe zones. Custodian will not kill / chase into harbors. Use them to plan.
- **Yellow laser alarms** — crossing fires a **security alarm** at that location and **forces investigation**. Recoverable mistake.
- **Headset mic** — talking near the Operator mic contributes to the noise model (EMCON lesson).
- **Scan / sprint / burst** — classic signature costs (unchanged).

## Run cards

### Alpha (~15 min)
Threat starts slow; goal = fuse 1 + return to harbor. Practice comms + map landmarks.

### Bravo (~25 min)
Full fuse run + generator + exit. Require at least one **harbor regroup**. Count laser alarms and burst sweeps as signature events.

### Charlie (~35 min)
MD may not read memo text (Operator reports intel). Emphasize initiative under compromise.

## Debrief questions

1. What was intent vs what happened?
2. Where did shared understanding break?
3. When did the Operator exercise initiative — was it aligned with intent?
4. Which decisions were reversible (harbor) vs irreversible (generator power-on / laser alarm)?
5. How did voice / scan / sprint trade against speed?

## Safety

- Stop word stops the exercise immediately.
- Use **Reduced Flash** on the boot screen if needed.
- Limit continuous VR to ~20–30 minutes; snap-turn only (no smooth turn).
- Horror intensity is intentional stress — instructors may pause the Custodian verbally (“freeze”) for teaching.
